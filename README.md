# Ibsidian

A modern, lightweight note-taking and knowledge management application with an Obsidian-inspired interface.

## Features

- **File Tree Navigation** - Browse and manage your notes and folders
- **Markdown Editor** - Write with syntax highlighting and live preview
- **Drawing Canvas** - Create visual notes with Excalidraw-style drawing
- **Terminal** - Built-in terminal for power users
- **Command Palette** - Quick access to all commands (Ctrl+K)
- **Resizable Layout** - Customize your workspace with draggable panels

## Run Locally

**Prerequisites:** Bun

```bash
bun install
bun run dev
```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- CodeMirror (editor)
- react-arborist (file tree)
- xterm.js (terminal)
